PharmaLedger: Blockchain-Based Pharmaceutical Supply Chain System

 Overview
PharmaLedger is a blockchain-based application designed to ensure secure, transparent, and tamper-proof tracking of pharmaceutical products across the supply chain. It helps prevent counterfeit drugs and improves trust among stakeholders.

 Key Features
🔗 Blockchain-based drug tracking (Ethereum)
🔐 Secure and immutable transaction records
📦 End-to-end supply chain visibility (Manufacturer → Distributor → Hospital)
📱 QR code-based drug verification
⚡ Real-time tracking and monitoring dashboard
🧾 Complete audit trail for every drug batch

🛠️ Tech Stack
Blockchain: Ethereum
Smart Contracts: Solidity
Backend: Flask (Python)
Frontend: HTML, CSS, JavaScript
Storage: IPFS (Decentralized Storage)
Tools: Web3.js, Ganache, MetaMask

🔄 Workflow
Manufacturer registers drug batch and generates QR code
Blockchain stores batch details as a secure record
Distributor verifies and updates ownership
Hospital performs final verification and confirms delivery
Every step is recorded as an immutable blockchain transaction

📊 Key Outcomes
Prevents counterfeit drugs using QR verification
Ensures data integrity through blockchain hashing
Provides real-time transparency across the supply chain
